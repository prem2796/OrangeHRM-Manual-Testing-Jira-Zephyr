# 🧪 OrangeHRM Manual Testing using Jira & Zephyr Scale

> An end-to-end Manual Testing project demonstrating Agile testing workflow using **Jira** and **Zephyr Scale**. This project covers test planning, test case design, execution, defect reporting, and QA documentation for the OrangeHRM application.

---

# 📌 Project Overview

This project demonstrates a complete **Manual Testing Life Cycle** for the OrangeHRM Human Resource Management System.

The objective was to simulate a real-world QA process by planning, designing, executing, and documenting manual test activities using industry-standard tools.

The project was managed using **Jira** for Agile project management and **Zephyr Scale** for test management.

---

# 🎯 Objectives

- Perform end-to-end manual testing of OrangeHRM.
- Validate critical business workflows.
- Create and execute professional test cases.
- Track defects using Jira.
- Maintain proper QA documentation.
- Demonstrate Agile testing workflow.
